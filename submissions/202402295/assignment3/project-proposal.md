# 개인 맞춤 뉴스 추천 봇

### 해결하고자 하는 문제
 하루에도 매우 많은 뉴스들이 쏟아져 나옵니다. 그 많은 뉴스들을 전부 확인하며 중요하거나 관심있는 뉴스를 골라내기는 거의 불가능합니다.

### 실제 사용 시나리오
 일정시간마다 새로 나온 뉴스들 중 사용자에게 중요하거나 관심있을 만한 내용을 가진 뉴스들의 제목과 요약, 링크를 디스코드 메세지로 받습니다.

### 제안하는 솔루션
 뉴스 API나 웹 크롤링을 통해 올라오는 뉴스들을 수집하고 뉴스의 카테고리, 제목, 내용 등을 고려하여 사용자에게 필요한 뉴스들의 제목, 요약, 링크를 디스코드 메세지로 보내는 봇입니다.

### 주요 기능:

 - 일정시간마다 새롭게 올라온 뉴스 자동 확인
 - 카테고리, 제목, 내용 등을 고려하여 사용자에게 필요한 뉴스 선별
 - 디스코드를 통해 해당 뉴스들의 제목, 요약, 링크를 전송
