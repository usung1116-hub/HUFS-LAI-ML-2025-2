# Assignment 4: Data Collection and Analysis

## 데이터 구성
- 총 200개의 영어 예문 데이터로 구성되어 있다.
- 데이터는 `sentence`(예문), `target_word`(타겟 단어), `label`(의미 분류) 총 3개의 주요 속성들로 구성되어 있다.
- 파일 경로는 `data/data.csv`이며, 결측값은 존재하지 않는다.

## 라벨 구성 및 특징
- **Target Words**: `address`, `account`, `present`, `contract`, `appreciate` 총 5개의 토익 빈출 다의어를 포함한다.
- **Labeling**: 각 단어는 문맥에 따라 두 가지 핵심 의미(0 또는 1)로 이진 분류되어 있다.
  - **Label 0**: 주로 명사(Noun) 또는 제1 의미 (예: 주소, 계좌, 선물, 계약서 등)
  - **Label 1**: 주로 동사(Verb) 또는 제2 의미 (예: 연설하다, 설명하다, 제시하다, 병에 걸리다 등)
- **클래스 분포**: 모든 타겟 단어에 대해 Label 0과 1이 각각 20개씩(총 40개) 균일하게 분포되어 있다.

## 데이터 분석 결과
- **데이터 균형성**
  - 라벨 간 분포를 확인할 때, 특정 클래스에 데이터가 쏠리지 않고 1:1 비율(50%:50%)을 완벽하게 유지하고 있다.
  - 클래스 불균형(Class Imbalance) 문제가 없어, 별도의 샘플링 기법 없이 모델 학습이 가능하다.

- **문장 길이 및 도메인 적합성**
  - 문장의 평균 길이는 약 6단어 내외로 분포되어 있다.
  - 이는 실제 토익 Part 5(단문 공란 메우기) 문제 유형과 유사하여 학습 데이터로서 적합하다.

## 데이터 품질 및 결론
- 결측치(Null)나 이상치(Outlier)가 발견되지 않아 데이터 품질이 양호하다.
