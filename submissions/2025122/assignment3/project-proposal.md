# 3차 과제 제안서 (2025122 최승택)

## 필요성

- arxivbot 이라는 프로젝트를 통해서 ai 논문의 한글 요약본을 받아보고 있습니다.
- 한글 요약으로 했지만, 하루에 200개씩 읽는건 너무 어렵습니다.
- 나와 관련있는 논문의 수는 200개 중에서 10개도 채 되지 않는 것 같습니다.
- 나와 관련없는 논문을 필터링 해줄 수 있는 text classification 모델을 붙이면 좋을 것 같습니다.

## 데이터 수집
- csv 에 논문의 제목 및 abstract 정보, 그리고 제가 직접 레이블링한 0/1 (relevance) 값을 함께 담을 예정입니다.
- irrelevant 한 논문 20개, relevant 한 논문 20개를 수집할 예정입니다.
