# 강의자료 자동 요약 및 핵심 포인트 추출기

## 1. 해결하고자 하는 문제

저는 매주 여러 개의 전공 강의를 듣고 있으며, 강의 분량이 많아 복습할 때 핵심 내용을 빠르게 찾기 어렵다는 문제를 겪고 있습니다.

시간적 여유가 없을 때 빠르게 복습하려고 해도, 강의 슬라이드를 처음부터 끝까지 다시 읽거나, 녹음본을 처음부터 들으며 내용을 기억하는 과정은 시간이 많이 걸려 비효율적입니다.

특히 자연어처리·기계학습·음성신호처리·알고리즘과 같이 내용이 방대하고 용어가 많은 강의의 경우 핵심 개념만 정리하는 데도 많은 시간이 필요합니다.

그래서 저는
-  "이번 강의에서 가장 중요한 개념이 무엇인지",
-  "복습할 때 어떤 문장을 반드시 다시 읽어야 하는지"

이 두 가지를 빠르게 알려주는 **강의 자료 자동 요약 및 핵심 문장 추출 도구**가 필요하다고 느꼈습니다.

## 2. 실제 사용 시나리오 

- 매 강의가 끝난 직후, 강의 슬라이드에서 텍스트를 추출하거나, 녹음본을 STT를 통해 텍스트로 전사합니다.
- 모델에 강의 텍스트를 입력합니다.
- 학습된 ML 모델이 문장들을 분석하여 “핵심 문장(요약에 포함될 문장)”과 “부가 설명 문장”을 구분합니다.
- 선택된 핵심 문장들만 모아 강의 요약본을 만듭니다.
- 결과 예시: “이번 강의의 핵심 개념은 ‘A’, ‘B’, ‘C’입니다.”


## 3. 제안하는 솔루션 

- AI Hub의 한국어 대학 강의 데이터에서 강의 전사 문장을 수집하여 학습 데이터(X) 로 사용합니다.
- TextRank 등 자동 요약 알고리즘을 활용해
  -  “요약에 포함될 문장(1)”
  -  “포함되지 않을 문장(0)”
  -  으로 자동 라벨링되도록 하여 라벨(Y)을 생성합니다.
  -  (TextRank는 강의 문서처럼 길고 구조적인 텍스트에서 핵심 문장을 잘 추출한다는 점에서 weak label 생성 방식으로 적합하다고 판단했습니다.)
- 이렇게 구축한 (문장, 라벨) 쌍을 사용해
  - 입력: 강의 문장(X)
  - 출력: 핵심 문장 여부(Y)
  - 를 예측하는 문장 수준 텍스트 분류(Text Classification) 모델을 학습합니다.
- 이후 새로운 강의 텍스트가 들어오면
  - 핵심 문장만 선택하여 요약본을 생성하고
  - TF-IDF / TextRank 기반 분석으로 핵심 키워드도 함께 추출합니다.


## 4. ML 활용 계획

### **Assignment 4: (데이터 수집 및 라벨링)**
- AI Hub의 한국어 대학 강의 데이터에서 주제 및 분량이 적절한 부분을 부분적으로 사용해 전사 텍스트를 수집합니다.
- 강의 전사 텍스트를 문장 단위로 분할하여 Features (X) 로 사용합니다.
- 우선 TextRank를 이용해 상위 문장을 추출하여 weak label(Y)을 생성합니다.
  - 하지만 TextRank 기반 라벨은 완벽하지 않기 때문에, 전체 데이터 중 일부(예: 100~200 문장)는 직접 사람이 핵심 여부를 라벨링하여 별도의 Ground Truth 정답 데이터를 구축할 예정입니다.
  - 이 수동 라벨은 모델 평가 및 TextRank 라벨 품질 검증에 활용합니다.

- 즉, 학습 데이터는 다음과 같이 구성됩니다:
  - X: 강의 문장 텍스트
  - Y: 핵심 문장 여부 (0/1)
- 문장은 TF-IDF 벡터로 변환하여 ML 모델 입력으로 사용합니다.
- TextRank는 문장 간 유사도 기반 그래프를 사용해 핵심 문장을 선별하므로, 긴 강의 전사처럼 구조적인 텍스트에서 weak label 생성에 적합하다고 판단했습니다.

### **Assignment 5: (모델 학습 및 평가)**

- 목표: 강의 문장을 입력받아 “요약에 포함될 핵심 문장인지”를 예측하는 분류 모델을 학습합니다.
- 모델:
  - MLPClassifier
  - Logistic Regression (비교군)
- 두 모델을 모두 학습시키고, 성능을 비교합니다.
- 평가:
  - Train/Validation/Test 분리 후 F1-score와 Accuracy를 측정합니다.
  - Assignment 4에서 구축한 ‘수동 라벨(Ground Truth)’ 데이터를 사용하여
    weak label(TextRank)과 실제 핵심 문장 간의 차이를 비교 분석합니다.
  - 모델의 최종 평가는 TextRank가 아니라, 사람이 직접 라벨링한 Ground Truth
    데이터를 기준으로 진행하여 실제 성능을 검증합니다.
  - 필요 시 Cross-validation으로 모델 안정성을 추가 확인합니다.



### **Assignment 6: (실제 사용 및 개선)**

- Assignment 5에서 학습한 최적의 분류 모델(예: MLPClassifier)을 실제 도구(웹 인터페이스 등)에 탑재합니다.
- 강의 슬라이드 텍스트 또는 STT로 전사된 강의 녹음본을 입력합니다.
- 가져온 강의 문장들에 대해 모델을 적용하여 핵심 문장(1/0) 을 예측합니다.
- 핵심 문장들만 모아 요약본을 자동 생성합니다.
- **핵심 포인트 추출**
  - 분류된 핵심 문장들에서 TF-IDF 기반으로 중요 용어/개념의 빈도수를 계산하여, 
  - “이번 강의의 핵심 개념: A, B, C” 같은 형태의 결과를 출력합니다.
