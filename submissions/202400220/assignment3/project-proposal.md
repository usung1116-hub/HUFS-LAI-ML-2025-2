# 개인화된 채팅 번역 도구

## 문제 제기
저는 Character.AI라는 서비스를 잘 사용합니다. 해당 서비스는 가상 캐릭터 등의 역할을 부여받은 AI와 대화를 주고받을 수 있는 챗봇 서비스입니다.

그러나 챗봇과의 대화는 영어만 지원되며, AI가 사용하는 문장에는 제가 이해하기 어려운 비유적 표현, 일상적 표현이 다수 존재합니다. 이 때문에 이해하기 어려운 표현이 등장 시 파파고로 번역해보지만, 특히 비격식적이고 생략된 표현은 번역이 전혀 이상하게 나오는 경우가 많습니다. 또한, 저도 영어로 대화를 진행해야 하는데, 말하고 싶은 표현이 영어로 정확히 떠오르지 않으면 또 파파고를 사용하고 제 문체에 맞게 수정하는 과정을 거쳤습니다. 그러나 이는 매우 번거롭고 시간 소모가 컸습니다. 그래서 이 문제를 해결하고자 개인화된 번역 도구를 제작해보기로 결심했습니다.

만약 잘만 만든다면 하루에 적어도 수십에서 수백(까지 가는지는 솔직히 잘 모르겠지만) 건의 대화 상황에 이 도구를 유용히 사용할 수 있을 것이라 확신합니다.

## 실 사용 시나리오
챗봇과 대화를 진행하다 이해하기 어려운 영어 표현이 등장하면 그 문장을 그대로 복사해서 붙여넣어 해석본을 받아봅니다. 만약 말하고 싶은 표현이 영어로 생각나지 않으면 한국어 표현을 작성하고 제 문체의 영어 표현을 받아봅니다.

## 제안하는 솔루션
이전까지의 대화 내역을 크롤링하거나 불가피할 경우 손수 카피하여 번역 데이터 쌍을 만들고, 이를 활용해 LM을 fine-tuning하여 개인화된 번역 도구를 제작합니다.
- SLM, sLLM, LLM 중 어떤 모델을 사용해야 비용 대비 효율이 좋을지는 아직 의문입니다.

## 주요 기능
- 내 문체에 맞춘 한영/영한 번역 제공
- 피드백 기능으로 직집 더 나은 번역을 제공하게 업데이트 가능
    - 번역 결과가 마음에 안 들면 직접 수정한 번역 결과를 데이터셋에 추가로 저장하여 파라미터 재업데이트
    - 구현 가능성이 의문이긴 합니다.

## ML 활용 계획

### Assignment 4
- 내 과거 대화 내역 데이터쌍  제작
    - 한 -> 영 100쌍: 내가 표현하려던 한국어 문장(새로 만들어야 함) + 내 문체의 영어 문장(대화 내역)
    - 영 -> 한 100쌍: 챗봇의 영어 문장(대화 내역) + 챗봇의 영어 문장을 번역한 한국어 문장(새로 만들어야 함)

### Assignment 5
- 번역 모델 학습
    - 데이터: 한영/영한 데이터 쌍 각각 100쌍씩 총 200쌍
    - 목표: 한국어 표현이 들어가면 내 문체의 영어로, 챗봇의 영어 표현이 들어가면 한국어로 번역되게 파인튜닝
    - 평가: BLEU score, 수동 평가
